安装配置mysql

我们一般用ZIP文件,直接配置就行了,解压完就是安装完.
官网下载界面,会看到两个,一个是带web-installer 这个表示需要联网安装,另一个不需要联网安装。
我们简单介绍下ZIP文件夹的内容
bin ：可执行文件存放
docs: 文档
include： .h头文件 (mysql是C语言写的)
lib: C语言库文件
share: 字符集相关信息
COPYING：版本信息
Readme： 开源协议
第一步：
配置环境变量： 为了方便在任何盘符下使用Bin文件下的可执行文件
%MYSQL_HOME% : D:\mysql-5.7.24
Path: %MYSQL_HOME%\bin
第二步：新建配置文件 my.ini (Home路径下)
字符集的信息和存储引擎的信息
[mysql]
default-character-set=utf8
[mysqld]
character-set-server=utf8
default-storage-engine=INNODB
sql-mode=STRICT_TRANS_TABLES,NO_ZERO_IN_DATE,NO_ZERO_DATE,ERROR_FOR_DIVISION_BY_ZERO,NO_AUTO_CREATE_USER,NO_ENGINE_SUBSTITUTION
注意下这里的txt文件要保存成utf-8 文件类型选*
第三步：初始化mysql
mysqld –initialize-insecure (管理员方式)
成功后不会返回信息。但在你的mysql home目录，会生成一个data文件夹，这就表示成功了(存放的mysql数据库信息)
第四步： 注册mysql服务
mysqld -install
（返回services成功安装），这个命令就是将mysql安装成windows service，这样开启就能启动了
第五步：
启动mysql
net start mysql // net stop mysql
到这个步骤，mysql其实已经完成了。这时候，我的电脑可以叫做mysql服务器
第六步：
修改账户名默认密码
mysqladmin -u root password demo12!@
